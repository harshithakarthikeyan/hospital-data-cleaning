## Project Overview
This project focuses on cleaning and preprocessing a hospital dataset using Python.
The goal was to prepare raw data for further analysis and machine learning tasks.

## Dataset
The raw dataset used in this project was synthetically generated for learning purposes and then cleaned and processed using Pandas.
The dataset contains hospital-related information such as:
- Patient details
- Admission and discharge data
- Missing and inconsistent values

## Data Cleaning Steps
- Removed duplicate records
- Handled missing values
- Corrected inconsistent data formats
- Renamed columns for clarity

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Outcome
A clean and structured dataset ready for analysis and modeling.
